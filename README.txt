Mediawiki-Template für die Seite: http://wiki.freifunk-franken.de/
Ursprünglich erstellt von https://github.com/melle/ffp-webdesign/tree/master/mediawiki-skin

Änderungen:

1. Änderungen am besten lokal auf dem eigenen Rechner vornehmen (z.B. im CSS).
2. Die Änderungen aus dem Ordner mediawiki-skin" auf den Server in das Verzeichnis "/web/mediawiki/skins" kopieren.
3. Falls die Datei".htaccess" geändert werden muss (ACHTUNG!) diese auf den Server in das Verzeichnis "/web" kopieren.
4. Änderungen online testen.
5. Wenn die Änderungen gut waren in git einchecken.
6. Wenn die Änderungen schlecht waren Änderungen rückgängig machen und Originale schnell wieder auf den Server spielen.

Änderungen bitte immer möglichst einzeln einchecken und sich am besten immer auf ein issue beziehen.

Bitte auf keinen Fall Dateien mit Passwörtern (z.B. LocalSettings.php) einchecken!
